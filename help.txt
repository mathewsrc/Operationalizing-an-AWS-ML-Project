Resources:

- Connect to remote machine shell:
	ssh -i <private-key-file>.pem user@<remote-ip>
	
- Download file to machine:
	wget https://video.udacity-data.com/topher/2022/June/62a388d0_starter/starter.zip

- Copy files from your machine to remote instance via SSH:
	scp -i <private-key-file>.pem file.zip user@<ip>:/remote/path/

- Copy a directory from your machine to remote instance via SSH:
	scp -i <private-key-file>.pem -r /path/to/local-dir/ user@<remote-ip>:/remote/path/

- Retrieve dog dataset images:
	aws s3 cp s3://udacity-aind/dog-project/dogImages.zip ./
	
  or:
  	wget https://s3-us-west-1.amazonaws.com/udacity-aind/dog-project/dogImages.zip
	
- List environments
	conda info --envs
	
- Activate pytorch environment on Amazon Deep learning Image (need to do this for your training script to work):

	source activate pytorch
	
- Launching Jupyter Lab on Amazon Deep Learning Image:

	jupyter lab

- Port forwarding to connect to jupyter lab
	ssh -i <private-key-file>.pem -L 8888:localhost:8888 <user>@<r
